# 钢铁雄心 4 模组管理器

_**通过使用 SteamCMD 的方式下载模组**_

## 食用方式

1. 从[发行版](https://github.com/Arama0517/hoi4-mod-manager/releases/latest)下载最新版本
2. 解压到游戏的根目录
3. 运行

**注意**: 本程序内置的共享账号可能会有时无法登陆
