# 钢铁雄心 4 模组管理器

_**通过使用 SteamCMD 的方式下载模组**_

## 食用方式

1. 从[Releases](https://github.com/Arama0517/hoi4-mod-manager/releases)下载最新版本
2. 解压到学习版的根目录
3. 运行本程序

**注意**: 本程序内置的共享账号可能会有时无法登陆

## 完成的功能

- [x] 基础功能
  - [x] 生成 Mod 引导文件 和 运行 dowser.exe
  - [x] 安装, 卸载, 更新模组
- [ ] 其他
